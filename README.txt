How it works:
Here is a sample code snippet to shuffle then remove random parts of a list of players:

players = [1, 2, 3, 4 ,5]
rnd = pytgm.players.random.seq.modify

rnd.shuffle(players)
rnd.remove(players, 3)

print(players)

The output could be:
[4, 2, 5]
